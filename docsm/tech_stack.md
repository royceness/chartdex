## Tech Stack

Use a simple, fast stack:

- Frontend: React + Vite + TypeScript + Tailwind
- Backend: FastAPI + Python
- Database: SQLite for app state and seeded metrics.  we should keep these databases separate
- Charts: Recharts
- Diagrams: Mermaid rendered in the browser (using an open source mermaid renderer)
- Voice: OpenAI realtime voice component - https://github.com/openai/realtime-voice-component
- Codex: codex app-server
